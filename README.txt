# 📝 Taskly

**Taskly** é um gerenciador de tarefas simples e eficiente feito com PHP e MySQL no backend e uma interface moderna usando HTML, Tailwind CSS e
JavaScript no frontend. Ideal para quem quer organizar sua rotina com tarefas diárias marcadas por horário.

---

## 🚀 Tecnologias utilizadas

- ✅ PHP (com MySQL)
- ✅ Tailwind CSS
- ✅ JavaScript (moderno, com fetch API)
- ✅ Vite (build frontend)
- ✅ HTML5
- ✅ CSS3

---
🖥️ Funcionalidades
 Cadastro e login de usuários

 Adição de tarefas com horário

 Concluir tarefas

 Deletar tarefas

 Interface responsiva

 Modo escuro

 Atualização em tempo real sem recarregar a página
